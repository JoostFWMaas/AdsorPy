# AdsorPy Roadmap

This document outlines the planned development path for **AdsorPy**, including short‑term improvements, medium‑term features, and long‑term research‑grade extensions.
The roadmap evolves as the project grows.

---

## 1. Short‑Term Goals

### Simulation Features
- Add **diffusion** of adsorbed species on the lattice
- Implement **desorption** to support reversible adsorption
- Introduce **species conversion** for on‑surface transformations

### User Experience
- Improve Python API ergonomics
- Add clearer configuration options for:
  - simulation modes
  - surface definitions
  - molecule creation

### Documentation
- Expand Sphinx docs with:
  - molecule‑building tutorials
  - simulation workflow examples
  - more visual diagrams
- Add more example scripts (e.g., multi‑species RSA)

---

## 2. Medium‑Term Goals

### Visualization
- Improve built‑in plotting utilities
- Add interactive visualization tools (widgets or browser‑based)
- Add a GUI for standalone execution

---

## 3. Long‑Term Goals

### Advanced Simulation Models
- Support heterogeneous surfaces with spatially varying rules

---

## Roadmap Status

This roadmap is a living document.
Contributions, suggestions, and feature requests are welcome through GitHub Issues and Discussions.
